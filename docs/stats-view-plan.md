# 统计视图方案

> 状态：Phase 1 ✅ 已完成（v0.8.x）  
> 进度与版本日志见仓库 Issues / Releases

## 目标

| 痛点 | 目标 |
|------|------|
| 年/月视图只能看「有没有写」，看不出「写了多少」 | 按月/按年汇总字数、篇数、活跃天数 |
| 缺少长期趋势感知 | 柱状图看月度波动，热力图看连续性（Phase 2） |

## Phase 1（已完成）

- 顶栏 `日历 | 统计` 切换
- 每年 3 指标卡：条笔记 / 字数 / 记录天数
- 展示全部有日记数据的年份
- 柱图 tooltip、点击月份跳转日历

## Phase 2–3（待做）

- GitHub 热力图
- 月/年粒度切换、多年柱状对比
- 统计范围可配置、CSV 导出、缺失提醒

详见源码 `src/services/StatsAggregator.ts`、`src/ui/StatsRenderer.ts`。
